## MiniSkidiESPNOW
Branch of ProfBoots MiniSkidi code to support a custom controller with control via ESPNOW wifi. The code uploaded here will be working code, so don't worry if there are few commits. I use this repo as a backup of known good states. I am not a coder, this is all generated with Gemini and checked by Codex.

#3D files
The original controller design can be found on OnShape [here](https://cad.onshape.com/documents/6bafb625ffbc5bb075599570/w/3596931d871cc26ccb4a224e/e/206217da13ada2750408ef04).

Printable files are [here](https://www.printables.com/model/1694698-miniskidi-controller), and I'll try to keep a list of components below.

#Components
Custom circuit board: KiCad files can be downloaded from Controller_V2 here. Ordered from JLCPCB, but PCBWay and others work too.
Joystick: [Analog 2-axis Thumb Joystick with Select Button + Breakout Board](https://thepihut.com/products/adafruit-analog-2-axis-thumb-joystick-with-select-button-breakout-board)
ESP32-C3 XIAO dev board: [Seeed XIAO ESP32C3](https://thepihut.com/products/seeed-xiao-esp32c3?variant=53975115694465)
Battery: [3.7V 600mAh Battery](https://www.amazon.co.uk/dp/B0DB1X3RK7)
Toggle switch: [5mm	toggle](https://www.amazon.co.uk/dp/B088R3GYWP)
Tacctile buttons: [PCB buttons, these or similar](https://www.mouser.co.uk/ProductDetail/Same-Sky/TS04-66-85-BK-160-SMT-TR?qs=VJzv269c%252BPaHwNn0xD14PA%3D%3D)
LED: [Neopixel LED](https://thepihut.com/products/neopixel-5050-rgb-led-with-integrated-driver-chip-10-pack)
Cap for NeoPixel: [0.1uF Capacitor](https://uk.farnell.com/vishay/k104k15x7rf5tl2/cap-0-1-f-50v-10-x7r/dp/2860097)
Light pipe:	[3mm, 1mm length](https://uk.farnell.com/mentor/1282-1405/light-pipe-1pipe-11mm-l-panel/dp/4562818)
